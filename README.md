# stevearchuleta.github.io

Personal portfolio website of **Steve Archuleta** — published at
[stevearchuleta.com](https://stevearchuleta.com).

A static, single-page site (HTML + CSS + a small amount of vanilla
JavaScript) covering data science, machine learning, financial
engineering, generative AI / NLP, and production MLOps — alongside the
story of a second career grown, quite literally, from the garden up.

## Contents

- **Story** — a short essay on retraining into AI/ML in a second career
- **Capstone** — the RiskML causal-aware risk-forecasting project
- **Selected Work** — four flagship MLOps and GenAI projects
- **About** — background, with hover-to-view credential certificates
- **CV** — two-page résumé, viewable in-page and downloadable as PDF
- **Contact** — links to email, GitHub, and LinkedIn

## Built with

- Hand-written HTML and CSS (no framework, no build step)
- Vanilla JavaScript for the CV modal and certificate hover images
- Google Fonts: Fraunces, Source Serif 4, IBM Plex Mono
- Hosted on GitHub Pages with a custom domain (see `CNAME`)

## Structure

```
.
├── CNAME              # custom domain configuration for GitHub Pages
├── index.html         # the entire site
├── README.md
└── assets/            # images, certificate PNGs, and the CV (PDF + DOCX)
```
